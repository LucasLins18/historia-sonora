# LIAD História Sonora — MVP

Aplicativo web simples para tocar efeitos sonoros por categoria, pensado para contadores de histórias.

## Estrutura

- `index.html` — página principal
- `styles.css` — estilos
- `app.js` — lógica global (volume, persistência)
- `pages/` — páginas internas por categoria
- `assets/sounds/{animals,nature,actions,fantasy}/` — arquivos mp3 (adicionar depois)
- `assets/images/` — logos

## Como usar

1. Coloque os arquivos de som nos caminhos indicados (nomes já definidos).
2. Abra `index.html` em um servidor (recomendado) ou localmente.
